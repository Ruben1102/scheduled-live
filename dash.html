DASH/ffmpeg

Put this file together with video.mkv in a directory, and serve that directory with any web server.
  • dash.js documentation
  • ffmpeg docs

Live Stream Audio and Video via DASH

ffmpeg \
    -re -i video.mkv \
    -strict -2 \
    -c:a opus -b:a 96k \
    -c:v copy -b:v 2000k \
    -f dash \
    -window_size 4 \
    -extra_window_size 0 \
    -min_seg_duration 2000000 \
    -remove_at_exit 1 \
    manifest.mpd
